Speak

The Speak command allows developers to prompt the vehicle's text-to-speech engine to speak text or phonemes to the head-unit.

In order to speak to the driver from your application, the following parameters are needed:

Text to Speak (required)

Type: String


This string represents the text (or phonemes) that should be spoken to the driver.

Speech Capability (required)

Type: Enumerated Value


This string represents the text (or phonemes) that should be spoken to the driver.

IMPORTANT NOTE: If the speech capability is SILENCE, the input text will represent the number of milliseconds the TTS engine should remain silent.

Available Speech Capability Options

Enumerated Value Speech Capability
TEXT The input string should be interpreted as raw text.
SAPI_PHONEMES The input string should be interpreted as phonemes from Microsoft's Speech API.
LHPLUS_PHONEMES The input string should be interpreted as phonemes from Nuance's API.
PRE_RECORDED The input string should be interpreted as a reference to a pre-recorded message that is supported by the head-unit. This option will vary per vehicle.
SILENCE The input text should contain the number of milliseconds of silence to play. Any speech that is in progress will be suspended for that amount of time.
